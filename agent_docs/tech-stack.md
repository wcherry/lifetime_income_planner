## Tech stack

### Framework & Runtime
- **Application Framework:** Actix Web
- **Authentication/Autherzation:** JSONWebToken (with Argon2 for password hashing)
- **OpenAPI Documentation:** Utopia
- **Linting/Formatting:** Clippy
- **Language/Runtime:** Rust
- **API Test Framework:** Playwright

### Frontend
- **Language:** Typescript 
- **JavaScript Framework:** React
- **UI Components:** Custom with Storybook
- **Package Manager:** Yarn
- **Test Framework:** Jest
- **End to End Test Framework:** Playwright
- **Linting/Formatting:** Prettier

### Database & Storage
- **Database:** SQLite
- **ORM/Query Builder:** Diesel
